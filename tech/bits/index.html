Bits

Posted on Jan 1, 2026

Basic operations

  • Get bit k
(x >> k) & 0x1
  • Set bit k
x |= (1 << k);
  • Clear bit k
x &= ~(1 << k);
  • Toggle bit k
x ^= ~(1 << k);

判斷 power of two

Level 1 — Interview 超常見

  • 判斷 power of two

(x > 0) && (x & (x-1) == 0)

  • Clear lowest set bit
x & (x-1)
  • 4-byte alignment:
(x + 3) & ~3
